PRESUPUESTO

Descripción de la propuesta

El usuario empieza el juego, luego el dinosaurio empieza a correr y debe evitar obstaculos, interactuando con la barra espaciadora (al estilo del dinosaurio de google) a medida que avanza el juego va subiendo el puntaje, y llegada determinada cantidad de puntos el juego le hace una pregunta al usuario para que pueda avanzar (si la responde correctamente), sino responde bien el juego vuelve a empezar y se reinicia la cantidad de puntos.

Alcance

Objetivos a realizar: Aguantar el maximo tiempo posible para conseguir la mayor cantidad de puntos.
